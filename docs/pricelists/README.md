# Прайсы поставщиков (локально, не в git)

Сюда кладём исходные файлы прайсов поставщиков для отладки парсера
(распознавание ценовых колонок, заголовки «с НДС»/«без НДС» и т.п.).

**Содержимое этой папки не коммитится** (см. `.gitignore`) — это
коммерческие данные поставщиков. Сами файлы прайсов в git не попадают,
трекается только этот README.

Штатный путь данных в БД — импорт через приложение (REPLACE), а не git.
